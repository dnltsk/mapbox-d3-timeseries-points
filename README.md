# mapbox-d3-timeseries-points
Timeseries data as D3 overlay on Mapbox

## environment
* npm v4.0.2
* node v7.2.0

## build
* `npm install`
* `npm run build`

## start
1. `npm run start`
2. open http://localhost:3000/
